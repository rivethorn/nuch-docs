Welcome to **NUCH**, a simple yet powerful CLI tool designed to streamline the management of Nuxt Content files and images, all from your terminal.

## What is NUCH?

NUCH is a CLI (Command-Line Interface) application built with Rust. It simplifies the process of managing Nuxt Content files and their associated images, ensuring that your content is always optimized and ready for deployment. With NUCH, you can easily publish new documents and delete obsolete files directly from your terminal, all while maintaining a clean and organized project structure.

## Key Features

This app includes a range of features designed to improve documentation management:

- **Powered by** [**Nuxt Content**](https://content.nuxt.com): Seamless file-based content management.
- **Built with** [**Rust**](https://www.rust-lang.org): High performance and efficiency.
- **Full file format support**: Markdown, JSON, YAML, and CSV.
- **MDC syntax** via Nuxt Content: Markdown with component integration for dynamic content.
- **Robust rollback mechanism**: Safely reverts changes to prevent data loss during content updates.
- **Safety first**: Prioritizes data integrity and security in all operations.
