Welcome to **NUCH**, a simple yet powerful CLI tool designed to streamline the management of Markdown content and images for your Nuxt Content sites.
## What is NUCH?

NUCH is a CLI (Command-Line Interface) application built with Rust. It simplifies the process of managing Markdown files and their associated images, ensuring that your content is always optimized and ready for deployment. With NUCH, you can easily publish new documentation and delete obsolete files directly from your terminal, all while maintaining a clean and organized project structure.



## Key Features

This app includes a range of features designed to improve documentation management:

- **Powered by** [**Nuxt Content**](https://content.nuxt.com): Utilizes the Nuxt Content module for seamless content management.
- **Built with** [**Rust**](https://www.rust-lang.org): Ensures high performance and efficiency in handling content operations.
- [**Markdown Support**]: Fully supports Markdown syntax for easy content creation.
- [**MDC Syntax**](https://content.nuxt.com/usage/markdown) **via** [**Nuxt Content**](https://content.nuxt.com): Supports Markdown with component integration for dynamic content.
- **Robust Rollback Mechanism**: Safely revert changes to prevent data loss during content updates.
- **Safty First**: Prioritizes data integrity and security in all operations, making sure your content is protected.
